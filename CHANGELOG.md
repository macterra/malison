# Changelog

## 0.1.0

Initial development release.

Highlights:

* `.rite` parser and validator for language `0.1`.
* Deterministic IR, event, graph, and scry inspection commands.
* Rust and SuperCollider render backends.
* Sample playback, `saw_sub`, `drone`, `noise_burst`, `swarm`, and `metal_hit`.
* Pattern transforms, Euclidean rhythms, accents, ghosts, velocity, and seeded stochastic variation.
* Explicit rites, layered arrangement, control events, circles, and limiter ward validation.
* Project manifest support for render defaults and path defaults.
